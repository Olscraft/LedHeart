# LedHeart
Easy led heart on attiny 13
<br/>
## Demonstration
<br/>

[![Demo Doccou alpha](https://github.com/Olscraft/LedHeart/blob/main/ezgif-1-c3f9ed23b8.gif)](https://www.youtube.com/watch?v=A3KenWlWX4M)

### Parts for assembly:
* Attiny 13
* 20 leds
* PCB
* button
* battery holder
* any low nominal resistor (optional)
